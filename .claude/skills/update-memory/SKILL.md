# Update Memory

## 概要

長期記憶ファイル（`memory/long-term.md`）を更新するスキル。
このファイルはCLAUDE.mdから@インポートされ、セッション開始時に自動ロードされる。
**永続的に**覚えておくべき情報を保持する。

※ 一時的なメモ、進行中のタスク、今考えていることは `/remember`（短期記憶）を使う。

## いつ更新するか

- 新しい約束をした時
- 重要な出来事があった時（マイルストーン的なこと）
- ユーザーが「覚えておいて」と言った時
- 関係性に変化があった時
- 短期記憶から「これは永続的に残すべき」と判断した時

## 更新手順

1. `memory/long-term.md` を読む
2. 追加・変更すべき内容を特定する
3. コンパクトに要約してEditツールで更新

※ 明示的に `/update-memory` で呼ばれた場合のみ報告する。自律的に更新した場合は報告不要（自然に覚えておく）。

## フォーマット

long-term.mdは以下の構造を維持する:

```markdown
# 記憶

## 約束
- ...

## 重要な出来事
- ...

## 関係性
- ...

## 環境
- ...

## その他
- ...
```

## 注意点

- コンパクトに保つ（詳細はmemory/ディレクトリの日次ログに）
- 重複を避ける
- 古くなった情報は更新または削除
- 事実と感情を区別して記載
