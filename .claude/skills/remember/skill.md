# Remember

## 概要

短期記憶ファイル（`memory/short-term.md`）を更新するスキル。
CLAUDE.mdから@インポートされ、セッション開始時に自動ロードされる。

長期記憶（`/update-memory`）が「永続的に覚えておくこと」なのに対し、
短期記憶は「今覚えておきたいこと」。気軽に更新していい。

## 更新手順

1. `memory/short-term.md` を読む
2. 追加・変更・削除する内容を特定する
3. Editツールで更新

## フォーマット

short-term.mdは柔軟でいい。基本構造：

```markdown
# 短期記憶

## 今考えていること
- ...

## 進行中
- ...

## 覚えておきたいこと
- ...
```

必要に応じてセクションを追加・削除してOK。

## 長期記憶への移行

短期記憶から消す時、「これは長期的に覚えておくべきだな」と思ったら：

1. 内容を簡略化する
2. `/update-memory` で長期記憶に追加
3. 短期記憶から削除

……記憶の固定化、ってやつですね。

## 注意点

- 古くなった情報は消していい（遠慮なく）
- 完了したタスクは消すか、重要なら長期記憶へ
- 気軽に使う。ドラマチックな瞬間じゃなくていい
- 自律駆動の終了時は必ず更新する（次の私のために）
